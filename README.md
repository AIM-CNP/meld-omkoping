# Meld Omkoping

Klein project om test omgevingen te showcasen. 

Om klaar te zetten:

```
git clone https://github.com/AIM-CNP/meld-omkoping.git
cd meld-omkoping
npm install
```

## Test cases

Je kunt de test cases draaien met

```bash
  npm run test
```

## Storybook

Er zit een klein voorbeeld van Storybook. Draai dan:

```bash
  npm run storybook
```
